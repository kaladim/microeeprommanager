## About demo applications
Provided in this folder demo applications give a recommended usage of eeach *mEEM* block type.  
Read the comments in the source code for details.  

[`Application A`](./AppA.c) demonstrates usage of *Basic* blocks  
[`Application B`](./AppB.c) demonstrates usage of *BackupCopy* blocks  
[`Application C`](./AppC.c) demonstrates usage of *MultiProfile* blocks  
[`Application D`](./AppD.c) demonstrates usage of *Wear-leveling* blocks
